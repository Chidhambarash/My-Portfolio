The Role

As a Buyer for Ola Electric, I was responsible for the end-to-end sourcing and vendor management for the Security Systems Integration (SSI) of the Ola Gigafactory. This project covered the complete physical security architecture of one of India’s largest EV manufacturing facilities, requiring a robust, IP-ready ecosystem.

Scope of Responsibility

CCTV Infrastructure

IP cameras, NVRs, and storage servers.

Access Control

Biometric/RFID, visitor management, turnstiles.

Networking Backbone

Cabling & active components for IP-readiness.

Perimeter Security

Boom barriers, vehicle bollards, fencing.

Command & Control

Control room video walls, VMS, alert automation.

Responsible for: Sourcing Strategy • Vendor Selection • Commercial Negotiation • Contract Finalisation

Strategic Supplier Management

Led the RFx cycle, technical evaluation, and TCO analysis with leading global OEMs and integrators:

Johnson Controls (JCI) Honeywell Bosch Security Hikvision Siemens

Collaborated with Indian system integrators (Inflow, Securens, SIS, Avalon) for on-ground execution.

Key Contributions & Achievements

Multi-Crore Project Execution

Executed end-to-end sourcing for a massive security integration covering 300+ cameras, 40+ access points, and a centralized command center.

10–12% TCO Savings

Negotiated savings through bundled procurement, multi-year AMC consolidation, and cross-leveraging suppliers from other Ola units.

Risk Mitigation & Compliance

Implemented dual-vendor strategies for critical components (redundant VMS, dual storage) and ensured 100% compliance with IT/OT policies.
